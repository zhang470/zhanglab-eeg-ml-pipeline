# 🧠 Zhang Lab EEG ML Pipeline

> ⚙️ Configurable, modular EEG classification framework  
> 📊 Featuring multiscale feature extraction & ensemble learning  

## Overview
**Zhang Lab EEG ML Pipeline** (*zhanglab-eeg-ml-pipeline*) is a flexible and extensible framework for EEG-based machine learning workflows.  
It supports modular pipeline design, multiscale feature engineering, and ensemble modeling for robust classification tasks.

## Background
This project is actively maintained by **Zhang Lab** as a continuation and extension of the original:

👉 https://github.com/yossefemara7/eeg_ml_pipeline  

The original version was developed by **Yossef Emara** during his time as a research assistant in Zhang Lab under the supervision of Dr. Yang Zhang.

## Key Features
- ⚙️ Modular pipeline architecture  
- 📊 Multiscale feature extraction  
- 🤖 Ensemble learning support  
- 🔄 Easy customization and extension  
- 🧩 Designed for research and reproducibility
